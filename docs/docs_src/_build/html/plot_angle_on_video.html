plot_angle_on_video module¶

plot_angle_on_video.lengthen_line(start_point, end_point, multiplier)¶

Takes in two points, and creates a lengthened version of the endpoint according to the multiplier. This is based on parameterized line.

Parameters

  • start_point – From point

  • end_point – To point

  • multiplier – Multiplier to lengthen the line by

Returns

Returns the lengthened version of the endpoint on the same line as the two original points

plot_angle_on_video.plot_angle_on_video(video_path, options, from_point, end_point_1, end_point_2, angles)¶

Plots the results from the analysis frame by frame onto the preprocessed video, with the given options.

Parameters

  • video_path – Takes inn the video path

  • options – Options: mode and focus

  • from_point – First point to plot from

  • end_point_1 – First point to plot to

  • end_point_2 – Second point to plot to

  • angles – Calculated angle for each frame
