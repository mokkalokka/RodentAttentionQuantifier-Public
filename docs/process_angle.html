process_angle module¶

process_angle.calculate_score(angles, from_index, to_index)¶

Takes in the angles and set the attention score according to the rats vision field.

Parameters

  • angles – Takes in the angles

  • from_index – From index (in case of multi-video analysis)

  • to_index – To index (in case of multi-video analysis)

Returns

Returns the attention scores

process_angle.get_angles(from_point, end_point_1, end_point_2, angles)¶

Takes in three points and creates a two vectors and calculates the angle between the two vectors.

Parameters

  • from_point – From point

  • end_point_1 – First destination point

  • end_point_2 – Second destination point

  • angles – Takes in the previous angles (in case of multi-video analysis)

Returns

Returns the calculated angles
