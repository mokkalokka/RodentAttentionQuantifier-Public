ma_dlc_controller module¶

ma_dlc_controller.analyze_videos(path_config_file, videofile_path, shuffle=1, videotype='mp4', manual_refinement=False)¶

Method for analysing the selected video with the pre-trained pose estimation model for DeepLabCut.

This method does the following pipeline:

  • DeepLabCut pose estimation

  • Converts the pose estimation to a raw tracklet file

  • Filters the predictions

Parameters

  • path_config_file – DeepLabCut config file

  • videofile_path – Video to be analysed

  • shuffle – Shuffle number

  • videotype – Type of video format (default = mp4)

  • manual_refinement – Possibility to refine the tracklets manualy (currently unused)

Returns

Returns the scorername that is used to open the pandas HD5 file
